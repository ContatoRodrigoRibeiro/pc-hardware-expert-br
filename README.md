# PC Hardware Expert BR - Versão 100% Local 🇧🇷

IA Especialista em Montagem de PCs **sem necessidade de API Key** — 100% baseada em dados reais do mercado brasileiro.

## Como usar (super simples)

1. `cd pc-hardware-expert-br-local`
2. `pip install -r requirements.txt`
3. `python run.py`
4. Abra o navegador em http://localhost:8501

Pronto! A aplicação já vem com dados reais de Abril/2026.

## O que tem dentro

- **Motor de recomendação local** inteligente (aloca orçamento de forma profissional)
- **Scrapers** para meupc.net, Kabum, Pichau e Terabyte
- **Banco de dados SQLite** com preços reais
- Interface Streamlit moderna e limpa

## Como atualizar preços

Clique no botão "Atualizar Banco de Dados (Scraping)" na barra lateral.
